## 🎨 Design System

### Color Palette
- **Primary**: Green 600 (#16a34a) - Trust, growth
- **Secondary**: Emerald 600 (#059669) - Nature
- **Accent**: Green 500 (#22c55e) - Energy
- **Feature Colors**: Blue, Purple, Orange, Teal
- **Neutrals**: Gray 900, 600, 50, White

### Typography
- **Headings**: Bold, 600-700 weight, -0.02em letter spacing
- **Body**: Regular, 400 weight, 1.6 line height
- **Sizes**: 64px (H1) → 12px (caption)

### Spacing
- **Base Unit**: 4px
- **Container**: 1280px max-width
- **Sections**: 96px vertical spacing
- **Cards**: 32px padding

### Components
- **Buttons**: Primary (gradient), Secondary (outline)
- **Cards**: Benefit, Feature, Dashboard
- **Icons**: 24px-64px from Lucide React
- **Shadows**: lg-2xl for depth
